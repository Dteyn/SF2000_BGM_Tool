How to Use Kerokero

Below are the basic steps on how to use Kerokero:

- Select your audio file
- Preview the audio until it gets to the point where you want to mark the start of the audio loop
- Click 'Mark In' to set the in point of the audio loop
- When the audio gets to the point where you'd like it to end, click 'Mark Out' to mark the end point
- Use the 'Preview Loop Transition' button to see how the loop will sound on the device
- Adjust the In and Out point as necessary until the loop is seamless
- If you wish, click 'Preview Full Audio Clip' to listen to the entire clip, looped 3x to hear how it'll sound
- When you're happy with it, click 'Save Audio Clip' to save a 'pagefile.sys' file
- Copy the resulting 'pagefile.sys' file to the '/Resources' folder on your SD card
